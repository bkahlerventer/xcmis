====== xCMIS Running procedure on Tomcat application server ======

1. Run "./xcmis.sh run" command within current /bin directory.
2. Available services URLs:
  2.1 Home page at the URL "http://localhost:8080/xcmis"
  2.2 REST Atom services will be available at the URL "http://localhost:8080/xcmis/rest/cmisatom"
  2.3 SOAP services should be available at the URL "http://localhost:8080/xcmis/cmisws"
3. Default service credential: username is "root" and password is "exo".
